## 이니셜 사용 규칙

> [!important] 이니셜 vs 풀네임
> - **이니셜** 사용: 반차, 40H OFF, 연차, 대휴, CT/MR → `종`, `승`, `동` …
> - **풀네임** 사용: 이브닝, 야간당직 → `이승남`, `김현석` …

이유: 이브닝/야간은 **성이 같은 직원이 있어서** 풀네임 필요
- 이승남 / 이동현 / 이용진 (이씨 3명)
- 송선경 / 송우석 / 송진우 (송씨 3명)
- 김종환 / 김현정 / 김현석 / 김봉선 (김씨 4명)

## 연차 계산 로직

```
사용가능 연차 = min(totalLeave, 20)
수당 전환분 = max(totalLeave - 20, 0)
잔여 연차 = 사용가능 - 사용일수
```

- 연차 20일 초과분은 **수당으로 자동 전환**
- 종일연차 = 1일 차감
- 반차(오전/오후) = 0.5일 차감
- 대휴/40H OFF = **연차 차감 없음**
