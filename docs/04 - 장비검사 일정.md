## 검사 주기 자동 계산

```javascript
// baseDate부터 cyclYears 간격으로 2035년까지 날짜 생성
function getInspectionDates(baseDate, cyclYears, maxYear = 2035) {
  const dates = [];
  const base = new Date(baseDate);
  for (let y = base.getFullYear(); y <= maxYear; y += cyclYears) {
    const d = new Date(base);
    d.setFullYear(y);
    dates.push(d.toISOString().slice(0, 10));
  }
  return dates;
}
```

예시: `mri-doc` (기준일 2026-04-02, 1년 주기)
→ `2026-04-02`, `2027-04-02`, `2028-04-02`, ... `2035-04-02`

## 검사일 조정 기능

- 기준일에서 **±21일** 이내로 이동 가능
- 이동 정보는 Firestore의 `inspDates` 키에 저장

```json
{
  "inspDates": {
    "2026-04-02_mri-precision": "2026-04-10",
    "2026-04-02_mri-doc": "2026-04-10"
  }
}
```

저장 키 형식: `{기준일}_{장비ID}` → 이동된 날짜

## 자동 공지 기능

D-15일 이내의 검사가 있으면 **공지바에 자동 표시**:

```
⚠️ 장비검사 예정
📅 D-3 🧲 MRI 정밀 정밀검사 (2026-04-02)
📅 D-3 📋 MRI 서류 서류검사 (2026-04-02)
```

캘린더 셀에는 🔴 빨간 `!` 아이콘으로 표시되며, 클릭하면 검사일 조정 팝업 오픈.
