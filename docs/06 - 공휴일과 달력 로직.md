## 달력 렌더링 로직

### 기본 흐름

```mermaid
flowchart TD
    A[renderCalendar 호출] --> B[cal-grid 초기화]
    B --> C[첫날 요일만큼 빈 셀 생성]
    C --> D[1일부터 말일까지 루프]
    D --> E{각 날짜별}
    E --> F[하이라이트 오버레이 체크]
    E --> G[공휴일 이름 표시]
    E --> H[교육일정 표시]
    E --> I[도트 마커]
    E --> J[배지 슬롯 렌더링]
    E --> K[이브닝/야간 하단 태그]
    D --> L[월 타이틀 업데이트]
```

### 셀 구조

```
┌─────────────────────┐
│ 5 💬 🔴              │  ← day-header-row (날짜 + 메모아이콘 + 검사아이콘)
│ 어린이날             │  ← holiday-name
│ 📚학술대회            │  ← edu-indicator
│ ●●                   │  ← dots (DOT_MARKERS)
│ ☀종 🌙승             │  ← badge-row (반차)
│ 40:동                │  ← badge-row (40H OFF)
│ 🏝선 🌿석             │  ← badge-row (연차/대휴)
│                      │
│ E:이동현             │  ← evening-tag
│ N:김현석             │  ← night-tag
└─────────────────────┘
```

## DOT_MARKERS (고정 도트)

특정 날짜에 보라색 도트를 표시하는 고정 데이터:

```javascript
const DOT_MARKERS = {
  "2026-03-07": 1, "2026-03-14": 2,
  "2026-03-21": 2, "2026-03-28": 1,
  "2026-04-04": 1, "2026-04-11": 2,
  "2026-04-18": 2, "2026-04-25": 1
};
```

> 도트 1개 or 2개로 표시. 아마 급여일/회의일 등의 고정 일정을 의미하는 것으로 추정.
