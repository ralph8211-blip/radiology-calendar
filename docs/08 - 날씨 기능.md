## 표시 정보

### 메인 날씨바

| 요소 | 내용 |
|------|------|
| 아이콘 | ☀️🌧⛈❄️🌫 등 날씨 이모지 |
| 현재 기온 | `26°` |
| 체감 온도 | `체감 28°` |
| 상세 | `구름 많음 💧65% 💨3m/s` |
| 출근 시각 | 07~09시 예보 아이콘 + 기온 |
| 퇴근 시각 | 16~18시 예보 아이콘 + 기온 |

### 5일 예보 (가로 스크롤)

요일, 날짜, 아이콘, 최고/최저 기온, 강수확률(≥20% 시)

### 날씨 배경 테마

| 날씨 코드 | 배경 그라디언트 |
|-----------|----------------|

| `02~04` (구름) | cloudy: `#4b6cb7 → #7b8fa1` |
| `09~10` (비) | rainy: `#373b44 → #4286f4` |

| `13` (눈) | snowy: `#a8c0ff → #3f2b96` |
| `01n` (맑은 밤) | night: `#0f0c29 → #302b63` |

## ⚠️ 퇴근길 알림 시스템

자동으로 감지하여 공지:

| 조건 | 알림 메시지 |
|------|------------|
| 퇴근 시 비 예보 | 🌂 퇴근길(17시경) 비 예보 — 우산 챙기세요! |
| 퇴근 시 천둥번개 | ⛈ 퇴근길 천둥번개 예보 — 조심하세요! |
| 퇴근 시 눈 | ❄️ 퇴근길 눈 예보 — 미끄럼 주의! |
| 오후 기온 8°이상 급강하 | 🧥 오후 기온 10°C 급강하 — 겉옷 챙기세요! |

> 알림은 빨간 펄스 애니메이션으로 강조 표시됨.
