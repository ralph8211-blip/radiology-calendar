# 🎨 UI 컴포넌트 가이드

## 화면 구성 (위→아래)

1. **NAV** — 로고 + 동기화 상태 + 로그아웃 + 정산 버튼
2. **MONTH HEADER** — ◀ 2026년 4월 ▶
3. **WEATHER BAR** — 현재 날씨 + 출퇴근 예보
4. **WEEKLY FORECAST** — 5일 예보 (가로 스크롤)
5. **NOTICE BAR** — 📢 공지 + 장비검사 알림
6. **CALENDAR GRID** — 7열 메인 캘린더
7. **BOTTOM BANNER** — 직원 선택 버튼 (fixed)

## 팝업/오버레이 5개

| 팝업 | ID | 트리거 | 애니메이션 |
|------|-----|--------|-----------|
| 로그인 | `#login-overlay` | 초기 진입 | shake (실패 시) |
| 날짜 편집 | `#popup-overlay` | 날짜 셀 클릭 | bottom sheet 슬라이드 |
| 대휴 사유 | `#alt-reason-overlay` | 대휴 배정 클릭 | scale 바운스 |
| 장비검사 조정 | `#insp-move-overlay` | 🔴 검사 아이콘 클릭 | scale 바운스 |
| 공지 편집 | `#notice-popup-overlay` | ✏️ 버튼 클릭 | translateY 슬라이드 |

## 디자인 패턴

- **Glassmorphism**: 배너, 오버레이에 `backdrop-filter: blur()` 사용
- **Border Radius**: `99px`(pill) / `2~3rem`(카드) / `1.2rem`(입력) / `4px`(배지)
- **팝업 바운스**: `cubic-bezier(0.34,1.56,0.64,1)` 스프링 효과
- **모바일 전용**: `max-width: 480px`, `user-scalable=no`
